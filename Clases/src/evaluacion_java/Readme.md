Comentarios del proyecto.

Para esta evaluacion se crearon las siguientes clases: 
	- Electrodomestico
	- Lavadora
	- Television
	- Ejecutable

Lavadora y Television son clases hijas de Electrodomestico, la clase llamada Ejecutable es nuestra clase principal.

Cree pruebas para las clases lavadora y television para comprobar el correcto funcionamiento de ellas, las cuales no arrojaron errores. Una vez terminada hice una prueba global para probar la conexion correcta de las clases y el funcionamiento modificando datos.

Para esta solucion se implemetó una clase principal (Ejecutable) en la cual se encuentra el arreglo con los 10 electrodomesticos y sus valores correspondientes, la clase Electrodomestico es la clase padre que extiede a las clases hijas lavadora y television por medio de los atributos precio base, color, consumo energetico y peso.

El precio final de todos los elementos (electrodomesticos incluidas las lavadoras y televisores) se logra por medio del metodo precioFinal que heredan todas las clases relacionadas padre e hijas. 

Realizo la sumatoria final, imprimiendo los valores solicitados.

Finalmente realicé un diagrama de clases en uml para tener una idea de la relacion que existe entre todas las clases del proyecto..
